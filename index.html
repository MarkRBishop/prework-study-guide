HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • There are several list types, ul, ol, and li.
  • 6 different types of headers h1 through h6
  • There is a head with meta data that isn't normally shown to the user and a body that encompasses what is shown on the page

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • Groups of elements or individual elements may be given attributes
  • If there is a specific thing you want to style, you may do it inline, which means that you can write it directly into the HTML

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pulls the main branch so your branch is up to date, can be any branch not just the main
  • git switch branch-name: changes to a branch if you have a separate branch from the main, but are not currently in it
  • git add -A: commits all the files in the working branch
  • git commit -m "description": adds a concise description of the changes made.
  • its a good habit to pull the origin branch before pushing an update to ensure they are synced

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript is an object oriented language, used to perform actions.
